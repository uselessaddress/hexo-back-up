# 毕设选题
想了很久，最终决定，做一个社区网站系统。主要有注册、登陆、发帖、回帖、关注、收藏、@、私信、分享、个人信息管理、系统管理等功能，参考 http://named.cn/.mine

用的最熟练的是Java，但是，这一次，小编决定使用不够熟练的技术——Node.js。因为，这是我未来想要深入发展的方向——前端！

这个项目可能用到的技术和框架有：Node.js、Express、EJS、Bootstrap、MongoDB。
