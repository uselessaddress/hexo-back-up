# 基础设置
1、显示类型
计算圆面积时，面积的结果显示，默认是short类型（保留小数点后4位），我们可以通过设置为long类型，来显示更精确的结果。
1）通过命令设置
`format long`，这种设置方式临时性的。

2）通过界面设置
File，Preferences，Command Window，Numeric format选择long，之后Apply即可。
